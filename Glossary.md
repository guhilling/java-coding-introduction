# Glossary

Quite a comprehensive online glossary can be found [here](https://stackify.com/java-glossary/).

Four our lectures, roughly in "order of appearance":

* Compiler: A program that reads source code and usually creates machine code.
  The Java compiler (``javac``) create bytecode instead that will be run by the:
* Java Virtual Machine: The program that "runs" the byte code (``java``).
* Source Code: The program code how we write and maintain it.
